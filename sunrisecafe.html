Sunrise Cafe

Sunrise Cafe is a coffee shop website I made for a fictional company.

About the Project

This was an individual project that required the integration of jQuery UI, AJAX/APIs, web storage solutions, and a dynamic slideshow/carousel to effectively display a variety of content.

This project was the culmination of my learning journey in a course focused on web development, specifically utilizing HTML, CSS, and JavaScript. It's designed to simulate a real-world scenario where I might need to showcase my skills to potential employers or clients. The assignment allowed for a creative demonstration of my ability to integrate various web technologies into a functional and aesthetically pleasing single-page application. Looking at the project requirements, I decided to make a coffee shop website since I thought immediately I could use a weather API to let users display the weather and recommend certain drinks based on the current temperature. I also wanted to use jQuery tabs for the menu display that allows the user to switch between different menu categories and see what's under each one.
